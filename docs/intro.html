Introduction¶

This is a set of utility functions to perform preliminary analysis on the NIX test data. However, the module is not specific to the instrument and is available for general use.

HDRL2 module provide python wrapper around some HDRL functions to use community proven algorithms such as Strehl ratio calculation. This feature requires a proper installation of CPL and GSL.

Install¶

To install:

git clone https://github.com/pssncp142/NIX_Testing.git
cd NIX_Testing
pip install .
